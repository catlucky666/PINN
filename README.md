# PINN
2025年上學期電腦課之期末專題

data.ipynb為生成資料程式，因為使用jupyter notebook編輯，副檔名為ipynb

data.csv為訓練用資料

test_data.csv為測試用資料

data.py與data,ipynb完全相同，特意複製一份是為了能在test.ipynb進行導入公式解之操作

ANN_model.ipynb為ANN模型架設、訓練檔案，PINN_model.ipynb同

ANN_test.ipynb為專門測試ANN模型用，PINN_test.ipynb同

ann_model.pth、ann_scaler.pkl為模型本身，pinn_model.pth、pinn_scaler.pkl同

PINN_model_7.ipynb，為第七次進行PINN訓練之檔案，因為變動幅度較大，所以獨立存放。其並非完成狀態，故將不會記錄相關資訊
